# 3dobject_tracking

Build instructions:
(From root directory of the repo, ie. /path/to/3dobject_tracking/)

```
mkdir build
cd build
```
Then:
(CMake):
```
cmake ../
make
```

or
(Ninja):
```
cmake -G Ninja ../
ninja
```

ROS version: http://wiki.ros.org/melodic/Installation (Recommended for Ubuntu 18.04 which is what I have)
